#  AI Sentiment Analysis Dashboard

An AI-powered Sentiment Analysis Dashboard that classifies text as **Positive** or **Negative** using Machine Learning. The application features a clean and interactive web interface built with Flask, allowing users to analyze reviews instantly.

---

##  Live Demo

 **Live Application:**  
https://sentiment-analysis-dashboard-yote.onrender.com

---

##  Project Overview

This project uses **Natural Language Processing (NLP)** and **Machine Learning** to predict the sentiment of user-entered text. A Logistic Regression model trained with TF-IDF vectorization analyzes reviews and returns the predicted sentiment along with the confidence score.

---

##  Features

-  Real-time sentiment prediction
-  Machine Learning based classification
-  Confidence score display
-  Fast and lightweight prediction
-  Modern and responsive user interface

##  Installation

Clone the repository

```bash
git clone https://github.com/rohanbhowm25308/Sentiment-Analysis-Dashboard.git
```

Move into the project directory

```bash
cd Sentiment-Analysis-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
